# Frontend Mentor - Body Mass Index Calculator

## Overview
This is a precise, responsive BMI (Body Mass Index) calculator. The application allows users to quickly determine their BMI and understand their weight category based on standard health metrics. While the design and assets were provided by Frontend Mentor, I built the entire functional application, including the unit conversion logic and dynamic result calculations, independently.

## Functionality
The application includes the following core features:

Unit Switching: Users can toggle between Metric (centimeters/kilograms) and Imperial (feet/inches/pounds) measurement systems.

Real-time Calculation: Automatically calculates the BMI score as soon as valid height and weight data are entered.

Weight Classification: Provides instant feedback on weight status (Underweight, Healthy weight, Overweight, or Obese) based on the following ranges:

Underweight: BMI less than 18.5

Healthy weight: BMI 18.5 to 24.9

Overweight: BMI 25 to 29.9

Obese: BMI 30 or greater

Healthy Range Guidance: Dynamically calculates and displays the ideal healthy weight range for the user's specific height.

Responsive Interface: A fully optimized layout that adapts seamlessly from mobile devices to large desktop screens.

Interactive States: Includes polished hover and focus states for all interactive inputs and buttons.
